# Docs Index

## Purpose
This index separates active documentation from archived run logs and defines a simple naming pattern so files are easier to scan.

## Naming Pattern
- `sprint{N}_{topic}_{type}.md`
- `{type}` should be one of: `log`, `decision`, `memo`, `rationale`, `manifest`, `experiments`.

Examples:
- `sprint4_stop_pivot_memo.md`
- `sprint5_tte_solidify_log.md`
